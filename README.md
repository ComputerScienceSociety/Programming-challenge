# Programming-challenge
This is the data-set for the programming challenge

# Questions
1) How many times does the word “spy” appear in TOP_SECRET.txt? 
2) How many lines in TOP_SECRET.txt contain a tab?
3) What is the longest repeated character sequence in TOP_SECRET.txt?  Give the character and the length of the sequence for both points.
4) In TOP_SECRET.txt what is the total of all of the numbers and what it the average value?  NB: “42” should be read as ‘4’ and ‘2’
5) In TOP_SECRET.txt what is the largest value that can be read?  EG: In the string “nyr8 3d29fd2” the largest readable value is 29.
